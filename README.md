 UnivioSA Web App

A Progressive Web App (PWA) that helps South African students explore universities, calculate APS, and track application deadlines.

Features

* APS Calculator
* University listings(statuses, opening and closing dates, applications fees, location, faculties etc)
* Application deadlines
* Prospectus downloads
* "Apply Now" links
* Bursary updates, news, tips
* In App notifications
* Buil-in chat system

Built With

* HTML
* CSS
* JavaScript
* Firebase(Authentication and Database)
* Cloudinary( Imaage and documents uploads)

Goal

To simplify university applications for students across South Africa.
